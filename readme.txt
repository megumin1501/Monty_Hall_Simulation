
# Monty_Hall_Simulation

A simulation of the Monty Hall problem implemented in Python



## About the project
It is a CLI program with 3 main features : 

+ Play the classic Monty Hall with 3 doors
+ Using Monte Carlo method to simulate Monty Hall problem, with customizable number of doors and turns
+ Visuazlize the win rate of "switch" and "stay" choice in line chart
